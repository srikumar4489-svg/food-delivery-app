# 🍔 Food Delivery App

A full-stack Food Delivery Web Application built using Django.

## 📌 Features

- User Registration & Login
- Restaurant Management
- Category-wise Food Listing
- Search Foods
- Add to Cart
- Place Orders
- Order Tracking
- Customer Reviews
- Responsive UI

---

## 🛠️ Tech Stack

- Python
- Django
- HTML5
- CSS3
- JavaScript
- Bootstrap
- SQLite

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/srikumar4489-svg/food-delivery-app.git
```

### Move into Project

```bash
cd food-delivery-app
```

### Create Virtual Environment

```bash
python -m venv .env
```

### Activate Environment

Windows

```bash
.env\Scripts\activate
```

### Install Packages

```bash
pip install -r requirements.txt
```

### Run Server

```bash
python manage.py runserver
```
